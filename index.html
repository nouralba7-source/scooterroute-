ScooterRoute NL

Navigeer via bromfietspaden en 50/60 km/h wegen. Eenmalige setup — daarna gewoon gebruiken.

1
Maak een gratis GraphHopper account
Ga naar graphhopper.com/dashboard en registreer. Gratis tier geeft 500 routes per dag.
2
Kopieer je API key
Na inloggen zie je jouw API key bovenaan het dashboard. Kopieer hem en plak hieronder.
API Key
3
Beveilig je key (optioneel maar slim)
In het GraphHopper dashboard → API keys → edit → voeg bij Allowed referrers je GitHub Pages URL toe, bijv. https://jouwusernaam.github.io/*. Dan werkt de key alleen op jouw site.
Waarom? Je API key staat in de broncode van de app. Met een referrer-restrictie kan niemand hem misbruiken — requests van andere sites worden automatisch geweigerd.
